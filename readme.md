## VMEX (Visualcamp Moblie Eyetracking Extension)

* **Xcode:** 9.3+
* **OS:** iOS 11+
* **Languages:** Swift 4.1+
* **Important:** This SDK is not built on the simulator but  only built on real devices.

## Getting started


### 1. TARGETS -> General -> Embedded Binaries -> Add VMEX.framework
